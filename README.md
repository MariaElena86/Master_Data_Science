🎓 Máster en Data Science e Inteligencia Artificial
 [Evolve Academy](https://evolve.es/)

## Misión:
La misión de este máster es ofrecer una formación práctica y transformadora que os permita no solo adquirir conocimientos teóricos, sino también aplicarlos en proyectos reales desde el primer día. Con nuestra metodología "Learn by Doing", aprenderás a manejar las herramientas más avanzadas del sector mientras trabajas en equipo y resuelves problemas reales. Aquí no nos quedamos en la teoría: aplicamos lo aprendido para que estéis preparados para enfrentar los desafíos del mundo laboral.

### Contenido:
- Bases de datos relacionales y no relacionales:
  - Trabajaremos con SQL para construir y gestionar queries avanzadas.
  - Exploraremos bases de datos no relacionales como MongoDB, esenciales para manejar grandes volúmenes de datos no estructurados.
- Programación en Python:
  - Tanto si ya tienes experiencia como si no, reforzaremos tus conocimientos.
  - Desde los fundamentos hasta el manejo avanzado de librerías como pandas, NumPy, y scikit-learn.
- Análisis y visualización de datos:
  - Utilizaremos herramientas como Power BI y Tableau para interpretar datos y crear informes que respalden decisiones estratégicas.
- Inteligencia Artificial y Machine Learning:
  - Aprenderás a construir modelos de machine learning, procesar lenguaje natural y explorar tecnologías emergentes como generación de imágenes y texto.

### Proyecto Final:
Desarrollarás un proyecto de principio a fin, aplicando todo lo aprendido para resolver un caso de negocio real. Este proyecto será evaluado por un jurado de expertos que podrían abrirte puertas a nuevas oportunidades laborales.
