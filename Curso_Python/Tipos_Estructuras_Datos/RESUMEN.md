
Estructuras de datos fundamentales del lenguaje. Comprenderlas y saber cuándo utilizarlas permite escribir código más limpio, eficiente y fácil de mantener.

🔹 Listas  
Colecciones ordenadas y modificables.

python
frutas = ["manzana", "banana", "uva"]
frutas.append("pera")
🔹 Tuplas  
Similares a las listas, pero inmutables.

python
colores = ("rojo", "azul", "verde")
🔹 Sets  
Colecciones sin duplicados y sin orden.

python
numeros = {1, 2, 3, 3, 4}
# Resultado: {1, 2, 3, 4}
🔹 Diccionarios  
Estructuras clave-valor para representar datos de forma clara.

python
persona = {"nombre": "Ana", "edad": 25}
💡 Cada estructura tiene su propósito. Elegir la adecuada según el contexto es clave para resolver problemas de forma eficiente.
